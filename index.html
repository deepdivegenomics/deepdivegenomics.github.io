About the Workshop

Deep Dive Genomics 2026 is a comprehensive, hands-on workshop designed for graduate students, early-career researchers, and bioinformatics enthusiasts who want to build a strong foundation in genomic data analysis. Over the course of this workshop, participants will gain practical skills in Linux command-line operations, variant calling pipelines using industry-standard tools, Python programming for biological data, and core algorithmic concepts that underpin modern bioinformatics workflows.

Whether you are just starting out in genomics or looking to solidify your computational skills, this workshop provides a structured, mentor-guided learning experience with real-world datasets and hands-on exercises.

What You Will Learn

  • ▶ Navigate and operate efficiently in a Linux environment for bioinformatics
  • ▶ Execute an end-to-end variant calling pipeline: from raw FASTQ reads to annotated VCF files
  • ▶ Write Python scripts to parse, analyze, and visualize biological data
  • ▶ Understand fundamental algorithms used in sequence alignment, genome assembly, and variant detection
  • ▶ Work with real-world genomic datasets and industry-standard tools (BWA, SAMtools, GATK, bcftools)

Who Should Attend?

This workshop is ideal for:

  • ▶ Graduate students in biology, bioinformatics, or computational biology
  • ▶ Researchers transitioning into genomics from wet-lab backgrounds
  • ▶ Anyone interested in learning the computational foundations of genomic analysis
