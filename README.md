# Deep Dive Genomics 2026

Workshop website for **Deep Dive Genomics 2026** — a foundational genomics and bioinformatics workshop covering Variant Calling Pipelines (with Linux), Python for Bioinformatics, and Algorithms in Bioinformatics.

🌐 **Live site:** [deepdivegenomics.github.io](https://deepdivegenomics.github.io)

## Courses

1. **Variant Calling Pipeline** — Linux, FastQC, BWA-MEM, SAMtools, GATK, bcftools
2. **Python for Bioinformatics** — Python fundamentals, Biopython, pandas, matplotlib
3. **Algorithms in Bioinformatics** — Dynamic programming, BWT, de Bruijn graphs, HMMs

- **Bikram** — Computational biology researcher, algorithms & genomic data science
