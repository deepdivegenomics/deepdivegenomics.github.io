Course 1: Variant Calling Pipeline

This module takes you through the complete variant calling workflow used in modern genomics laboratories. Starting with an introduction to Linux and the command line, you will learn to navigate filesystems, write shell scripts, and use essential bioinformatics CLI tools. The pipeline covers: quality control of raw sequencing reads (FastQC, Trimmomatic), read alignment to a reference genome (BWA-MEM), post-alignment processing (SAMtools, Picard), variant calling (GATK HaplotypeCaller), and variant filtering and annotation (bcftools, SnpEff).

Topics covered:

  • ▶ Introduction to Linux & the command line
  • ▶ FASTQ format and quality control
  • ▶ Reference genomes and indexing
  • ▶ Read alignment with BWA-MEM
  • ▶ SAM/BAM file processing with SAMtools
  • ▶ Variant calling with GATK
  • ▶ VCF format, filtering, and annotation

Course 2: Python for Bioinformatics

This module introduces Python programming with a focus on biological data analysis. No prior programming experience is required. You will learn Python fundamentals — variables, data structures, control flow, and functions — and then apply them to parse common bioinformatics file formats (FASTA, FASTQ, GFF, VCF), automate repetitive analysis tasks, and create publication-quality visualizations using matplotlib and seaborn.

Topics covered:

  • ▶ Python basics: variables, loops, conditionals, functions
  • ▶ Working with strings and sequences
  • ▶ File I/O: reading and writing biological data files
  • ▶ Biopython: sequence manipulation and BLAST integration
  • ▶ Data analysis with pandas
  • ▶ Visualization with matplotlib and seaborn

Course 3: Algorithms in Bioinformatics

This module covers the fundamental algorithmic concepts that power bioinformatics tools. You will study sequence alignment algorithms (Needleman-Wunsch, Smith-Waterman), understand the Burrows-Wheeler Transform and FM-index used in modern read aligners, explore graph-based approaches to genome assembly (de Bruijn graphs), and learn the statistical models behind variant calling (Bayesian genotyping).

Topics covered:

  • ▶ Dynamic programming for sequence alignment
  • ▶ Global vs. local alignment
  • ▶ BLAST heuristics and seed-and-extend
  • ▶ Burrows-Wheeler Transform (BWT) and FM-index
  • ▶ de Bruijn graphs for genome assembly
  • ▶ Hidden Markov Models in bioinformatics
  • ▶ Bayesian approaches to variant calling
